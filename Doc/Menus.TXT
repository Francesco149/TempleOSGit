A menu appears when you move the mouse to the top of the screen.  Menus are 
created with [C:/Adam/Menu.CPP.Z,152] MenuPush(), [C:/Adam/Menu.CPP.Z,160]
MenuFilePush(), [C:/Adam/Menu.CPP.Z,83] MenuNew() or [C:/Adam/Menu.CPP.Z,103]
MenuFile() and assigned 
to Fs->cur_menu.  The format is:


File 
{
  Open(,'O');
  Save(,'S');
  Exit(,CH_SHIFT_ESC);
}
Edit {
  Cut(,,SC_DELETE|SCF_SHIFT);
  Paste(,,SC_INS|SCF_SHIFT);
}
Misc {
  Opt1(MSG_CMD,M_OPTION1);
  Opt2(MSG_CMD,M_OPTION2);
}
Help
{
  Help(,'?');
  About(,'/');
}

The first arg is the msg code and it is optional with the default being 
[C:/Kernel/Adam1a.HPP.Z,2999] MSG_KEY_DOWN_UP.  The second arg is the msg p1
value which is ASCII of the key 
in the case of [C:/Kernel/Adam1a.HPP.Z,2984] MSG_KEY_DOWN.  The third arg is
the msg p2 value which is the S
can Code of the key in the case of [C:/Kernel/Adam1a.HPP.Z,2984] MSG_KEY_DOWN.

See [C:/Demo/PullDownMenu.CPP.Z,1].
