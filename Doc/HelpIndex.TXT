+] DolDoc
    [C:/Doc/DolDocOverview.TXT.Z,1] DolDoc is the propriatary TempleOS
    document type.  The editor, compiler, assembler and whole operating system
    seamlessly handle DolDocs.  Cmds like [C:/Kernel/Dsk/Dsk1b.CPP.Z,234] Dir()
    and [C:/Adam/Utils/Grep.CPP.Z,143] Grep() can output links to the cmd line.
  +] Burning CD/DVDs
    * Prepare a directory with the files you wish to burn.

* Use [C:/Adam/Boot/DskCDDVD.CPP.Z,326] CDFileCreate() to make an ISO image
    file.

* Use [C:/Kernel/Dsk/Dsk1c.CPP.Z,259] CDImageWrite() to burn an ISO file onto
    a CD or DVD.

* See [C:/Misc/MakeDistro.CPP.Z,1] Making a Distro ISO.
  +] Level 2: Admin Help
  Use [C:/Kernel/Dsk/DskAddDev.CPP.Z,173] Mount() to mount disk drives,
  perhaps, if you are booted to a LiveCD.

You recompile the kernel with [C:/Adam/Boot/InsBoot.CPP.Z,130]
  MakeOSDrvBootInstall(), and specify which drives should be mounted at boot. 
  See [C:/Misc/MakeDistro.CPP.Z,1].
