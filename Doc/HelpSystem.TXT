#help_index "Topic/SubTopic;OtherTopic"
The help index preprocessor compiler directive sets the topics for syms 
subsequently defined.  You specify subtopics with a '/' tree hierarchy and 
multiple topics separated by a ';'.  The index ctrls HI:topic links.

public can be placed before symbol to make it show.

#help_file "filename[.TXT.Z]"
The help file preprocessor directive places a file as the heading text in a HI
:topic link for the current help index.
