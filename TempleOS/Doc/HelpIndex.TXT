+] DOLLAR SIGN CMDS
    
[C:/TempleOS/Doc/Dollar.TXT.Z,1] Dollar Sign Cmds are rich-text cmds.  The
    editor, compiler, assembler and operating system are seamlessly equiped to
    handle TempleOS rich text.  Cmds like
    [C:/TempleOS/Kernel/Dsk/Dsk1b.CPP.Z,235] Dir() and
    [C:/TempleOS/Adam/Grep.CPP.Z,163] Grep() can output links to the cmd line.

  +] CMD PMT
    
The TempleOS cmd line interface is based on C/C++. Explore the ".CPP.Z"
    files.  A cmd which takes no args may be executed by typing the name and a
    semicolon.  Cmds with args require placing the arg in parentheses like a
    function call.

Dir("*");
`Dir("*");`
 will list files in the current directory.

Dir;
`Dir;`
 uses default of "*".

Cd("::/TempleOS/Apps");
`Cd("::/TempleOS/Apps");`
 will change directories.

Directories are referenced with "/" not "\".  The boot drive is referred to
    with "::".  Other drives are referred to by a letter and a colon.  Your user
    acct directory is referred to with "HOME".

If it is not found, ".Z" is added or removed and a search is done, again.  If
    a file is still not found, all parent directories are searched.


Ed("FILENAME.TXT.Z"); Invokes the editor. See
    [C:/TempleOS/Adam/Doc/Doc.CPP.Z,44] DOC LINK TYPES.


Some cmds like [C:/TempleOS/Adam/Grep.CPP.Z,163] Grep() take a
    "find_files_mask".  They begin with the name of a dir tree and end with a
    list of filename masks.  See [C:/TempleOS/Kernel/Dsk/DskStr.CPP.Z,40]
    FilesFindMatch().

Grep("pattern","/TempleOS/Demo/*.CPP.Z;*.TXT.Z;");

Be aware of [C:/TempleOS/Kernel/Adam1a.HPP.Z,2212] FILENAME_MASK_TXT and
    [C:/TempleOS/Kernel/Adam1a.HPP.Z,2211] FILENAME_MASK_SRC.

  +] BURNING CD-ROMs/DVDs
    
0) Prepare a directory with the files you wish to burn.
1) Use [C:/TempleOS/Adam/Boot/DskCDDVD.CPP.Z,328] CDFileCreate() to make an
    ISO image file.
2) Use [C:/TempleOS/Kernel/Dsk/Dsk1c.CPP.Z,306] CDImageWrite() to burn an ISO
    file onto a CD or DVD.

Or, see [C:/TempleOS/Demo/MakeDistro.CPP.Z,1] Making a Distro ISO.
